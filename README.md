# OOP

This repository contains a collection of Object-Oriented Programming projects and lab exercises demonstrating practical use of core OOP principles such as encapsulation, inheritance, polymorphism, and abstraction.

## Overview

Each subfolder represents an individual project or lab assignment focused on applying object-oriented design, building reusable components, and improving software architecture skills.
